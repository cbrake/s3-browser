s3-fetcher
==========

Command line app to fetch files from s3-browser server.

Installation
------------

If you have a $GOPATH set up:

`go get github.com/cbrake/s3-browser/s3-fetcher`

To build manually:

`go build s3-fetcher.go`

Usage
-----

```
s3-fetcher usage:
  -f string
        url of file to download
  -p string
        password
  -u string
        user name
```
